# 외부 안전 접속 초안

1단계에서는 공개 인그레스를 열지 않는다. 외부 접속은 후속 단계에서 구현하며, 기본 권장안은 `Tailscale` 기반 사설 네트워크다.

## 권장 방향

- 웹과 API는 기본적으로 로컬 바인딩 상태로 운영한다.
- 외부 조회가 필요할 때만 Tailscale 노드에 연결한다.
- 실전 모드와 외부 접속은 같은 단계에서 열지 않는다.

## 후속 체크리스트

- 운영 기기와 원격 기기의 Tailscale 인증 정책
- 브라우저 접근 시 추가 인증 계층 필요 여부
- 실전 계좌 사용 시 비밀값 저장 방식
- 원격 접속 중 긴급 중지 경로 확보
