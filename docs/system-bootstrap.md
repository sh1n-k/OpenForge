# 시스템 부트스트랩 구조

## 모노레포 기준

- 웹과 API는 같은 레포에서 관리한다.
- 프론트는 상태 조회와 운영 콘솔 셸만 담당한다.
- 백엔드는 버전 고정된 `/api/v1` REST 인터페이스와 DB 마이그레이션을 담당한다.

## 1단계 범위

- `GET /api/v1/health` 공개
- Flyway 기반 공통 스키마 생성
- `app_config`, `app_event_log` 초기 테이블 생성
- 프론트 루트 화면에서 백엔드 상태 표시

## 비범위

- 계좌 연결
- 전략 저장
- 외부 공개 접속
- 인증/권한
